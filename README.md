# 🎮 Kona Games

## هيكل المشروع
```
kona-games/
├── server.js              ← السيرفر الرئيسي
├── package.json
├── تشغيل.bat              ← شغّل من هنا (Windows)
├── data/
│   └── questions.json     ← كل الأسئلة (11 فئة، 600+ سؤال)
├── public/
│   ├── index.html         ← صفحة اختيار الفئات
│   ├── game.html          ← لوحة اللعبة
│   ├── question.html      ← شاشة السؤال
│   ├── end.html           ← شاشة النهاية
│   ├── css/style.css      ← التصميم
│   └── js/
│       ├── setup.js
│       ├── game.js
│       └── question.js
└── admin/
    └── index.html         ← لوحة الأدمين

```

## 🚀 تشغيل المشروع
1. تأكد Node.js مثبت من https://nodejs.org
2. انقر مرتين على `تشغيل.bat`
3. المتصفح يفتح تلقائياً

## 🌐 الروابط
- اللعبة: http://localhost:3000
- الأدمين: http://localhost:3000/admin

## 🎮 كيفية اللعب
1. اختر 6 فئات من القائمة
2. أدخل اسم اللعبة والفريقين
3. اضغط "ابدأ الآن"
4. اختر سؤالاً من اللوحة
5. اقرأ السؤال - المؤقت يصعد تلقائياً
6. اضغط "اظهر الجواب"
7. اختر من أصاب - تُضاف النقاط تلقائياً
8. في النهاية يظهر الفائز 🏆
